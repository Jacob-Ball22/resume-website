# Website Features & Functionality

## Core Features (Required)

### 1. Navigation
- [ ] Sticky header that hides on scroll down, shows on scroll up
- [ ] Smooth scroll to sections when clicking nav items
- [ ] Mobile hamburger menu
- [ ] Active section highlighting in nav
- [ ] Logo/name that returns to top when clicked

### 2. Hero Section
- [ ] Animated headline with typewriter or fade-in effect
- [ ] Professional title/tagline
- [ ] Call-to-action buttons (Contact Me, Download Resume)
- [ ] Social media links (LinkedIn, GitHub, etc.)
- [ ] Optional: Profile photo or avatar

### 3. Experience Section
- [ ] Timeline layout showing career progression
- [ ] Expandable job descriptions
- [ ] Company names and dates
- [ ] Key achievements with bullet points
- [ ] Technology tags for each role

### 4. Education Section
- [ ] Degree information with graduation dates
- [ ] Relevant coursework (collapsible)
- [ ] GPA if above 3.5
- [ ] Certifications with verification links

### 5. Skills Section
- [ ] Categorized skills (Frontend, Backend, Tools, etc.)
- [ ] Visual skill level indicators (bars, dots, or percentages)
- [ ] Search/filter functionality for skills
- [ ] Hover effects showing years of experience

### 6. Projects Section
- [ ] Project cards with images/screenshots
- [ ] Technology stack badges
- [ ] Links to GitHub repos and live demos
- [ ] Brief descriptions with key features
- [ ] Optional: Case study links for detailed write-ups

### 7. Contact Section
- [ ] Contact form with fields:
  - Name (required)
  - Email (required, validated)
  - Subject (optional)
  - Message (required, min length)
- [ ] Form validation and error messages
- [ ] Success confirmation after submission
- [ ] Alternative contact methods displayed

### 8. Footer
- [ ] Copyright information
- [ ] Quick links to main sections
- [ ] Social media icons
- [ ] Last updated date
