# Resume Website Template

This template helps you create a professional resume website similar to Jerome Benton's site using Claude Code.

## Quick Start

1. Copy all files from this directory to your project
2. Update the content in each markdown file with your information
3. Follow the instructions in CLAUDE.md to build your site

## Files Included

- `GETTING_STARTED.md` - **START HERE!** Step-by-step guide for VS Code and Claude Code
- `CLAUDE.md` - Instructions for Claude Code to build your site
- `CONTENT.md` - Your resume content (education, experience, skills, etc.)
- `DESIGN.md` - Design specifications and styling preferences
- `FEATURES.md` - Website features and functionality requirements
- `SAMPLE_COMMANDS.md` - Example commands to use with Claude Code

## How to Use with Claude Code

1. Open Claude Code in your project directory
2. Tell Claude: "Build me a resume website using the specifications in the markdown files"
3. Claude will read the files and create your website
4. Test locally and deploy when ready

## Customization

Feel free to modify any of the markdown files to match your preferences:
- Change colors and fonts in DESIGN.md
- Add or remove sections in CONTENT.md
- Request additional features in FEATURES.md
