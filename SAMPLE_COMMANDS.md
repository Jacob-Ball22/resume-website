```
"Create a Next.js resume website using the specifications in the markdown files"
```

## Tips for Your Nephew

1. **Start Simple**: Begin with basic sections and add features gradually
2. **Test Often**: Check your work in the browser frequently
3. **Be Specific**: The more detailed your instructions, the better the results
4. **Iterate**: Don't hesitate to ask Claude to refine or adjust things
5. **Save Progress**: Commit your changes to Git regularly
